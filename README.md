# Car Parts Search

Веб-приложение для поиска автозапчастей с отображением информации о производителе.  
Используется публичный API NHTSA VPIC.

## Демо

[Открыть сайт](https://EkateRazuyeva.github.io/car-part-search)

## Функционал

- Поиск автозапчастей по названию
- Просмотр информации о производителе в модальном окне
- Фильтрация по типу запчасти
- Пагинация результатов
